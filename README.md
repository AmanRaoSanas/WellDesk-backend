# WellDesk Platform Monorepo

This repository contains all code for the WellDesk project:

- **desktop-app/**: .NET Desktop Application
- **api-service/**: FastAPI backend service
- **cloud-etl/**: AWS ETL Python service
- **onboarding-website/**: React/Next.js frontend for user onboarding
- **shared-utils/**: Optional shared utilities and libraries

## Getting Started

- Clone this repo
- Navigate to each folder to setup & run individual services
- Each service has its own README.md for detailed instructions
